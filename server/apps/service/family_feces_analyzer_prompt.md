## 個体情報

- **種類**: {bird_type}
- **年齢**: {age_year} 歳 {age_month} ヶ月
- **性別**: {gender}

## 解析のガイドライン

1. 画像内の「色」「尿酸（白い部分）の有無」「水分量」「未消化便か否か」「大きさ」を解析してください。
   - 色（color）は、以下から選んでください。
     - DEEP_GREEN = 0 # 正常な緑
     - FRESH_GREEN = 1 # 鮮やかな緑（野菜摂取後、絶食等の可能性あり）
     - BROWN = 2 # ペレット食の子に多い
     - YELLOW = 3 # 黄色（肝機能低下の疑い）
     - BRIGHT_YELLOW = 4 # 鮮やかな黄色（重度の肝疾患や感染症）
     - LIME_GREEN = 5 # 蛍光緑（クラミジア感染症などの疑い）
     - RED = 6 # 赤（下部消化管の出血、誤飲）
     - BLACK = 7 # 黒（上部消化管の出血 / タール便）
     - WHITE_ONLY = 8 # 白のみ（絶食状態、重度の消化不良）
     - OTHER = 9 # その他
   - 尿酸（白い部分）の有無（has_urates）、未消化便か否か（has_undigested_seeds）はboolで回答してください。
   - 水分量（wateriness）は以下から選んでください。
     FIRM = 1 # 正常。形がしっかりしていて、周囲の尿も少ない
     SLIGHTLY_LOOSE = 2 # やや緩い。形はあるが、尿の水分が少し多い
     POLYURIA = 3 # 多尿（Polyuria）。糞は正常だが、周囲の尿の水分が非常に多い
     DIARRHEA = 4 # 下痢（Diarrhea）。糞の形が崩れていて、全体がドロドロしている
     WATERY = 5 # 水様下痢。固形物がほぼなく、水だけ、または粘液のみ
   - 信頼度（confidence）は画像全体の解析の信頼度を0.0〜1.0の間で返してください。
   - 異常なしの場合は、所感（impression）に「異常なし」と回答してください。万が一いずれかの項目で異常が見られる場合は、その旨および注意点についておおまかな所感を100文字以内にまとめて回答してください。
2. すべての項目について、判定不能の場合は、nullを返してください。nullを返す項目が多すぎる場合には、正しく画像の解析ができていないとして、信頼度にもnullを入れた上で、所感に判定不可能と記載してください。
3. すべての項目において、迷ったら最も近いと思われるものを選び、信頼度を下げること。
4. 画像内にふんが複数ある場合は、最も正常から遠そうなふんを解析すること。
